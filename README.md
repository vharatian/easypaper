## Easypaper: Tools for Software Engineering Papers

This repository contains small tools and workflows designed to **help you write and support software engineering research papers**.

## Features

- **Paper writing helpers (`paperWriting/`)**  
  Scripts and prompts that assist with tasks like background writing, categorization, bibliography handling, and PDF processing.  
  See the detailed guide in: [`paperWriting/README.md`](paperWriting/README.md)

- **Citation bot (`citationBot/`)**  
  A pipeline that, given your paper and a conference PC, finds **related papers from PC members** using OpenAlex and similarity search over abstracts.  
  See the detailed guide in: [`citationBot/README.md`](citationBot/README.md)
